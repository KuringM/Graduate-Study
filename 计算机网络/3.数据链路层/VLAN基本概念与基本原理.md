# VLAN基本概念与基本原理

由IEEE 802.1Q 工作组负责

大型局域网面临的问题

- 整个局域网是一个很大的广播域, 容易出现**广播风暴**
- 不利于信息安全

VLAN的特点

- 可将一个大型局域网分割成若干个较小的VLAN, **每个VLAN是一个广播域**
- 需要使用支持VLAN功能的以太网交换机来实现
- 每个VLAN对应一个 **VID**
