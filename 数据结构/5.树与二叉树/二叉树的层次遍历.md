# 二叉树的层次遍历

![[截屏2025-09-01 14.41.00.png]]

算法思想:

1. 初始化一个辅助队列;
2. 根结点入队;
3. 若队列非空, 则队头结点出队, 访问该结点, 并将其左、右孩子插入队尾(如果有的话);
4. 重复3直至队列为空.

typedef struct BiNode{
	char data;

	BiNode * data;//存指针而不是结点
